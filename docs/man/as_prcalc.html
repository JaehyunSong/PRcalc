    Transformation from data.frame to prcalc

    Description

    Transformation from data.frame to prcalc

    Usage

    as_prcalc(x, l1, l2 = NULL, p, q = NULL, type = c("party", "district"), ...)

    Arguments

    x a data.frame or tibble object.
    l1 a character (mandatory). A column name of region or state.
    l2 a character (optional). A column name of party or district.
    p a character (mandatory). A column name of population.
    q a character (optional). A column name of magnitude. If NULL, magnitudes of all districts are replaced by 1.
    type a type of l2. If “party”, vector of l2 represents parties. If “district”, vector of l2 represents electoral district.
    … ignored

    Details

    an object x must include three columns—region(state) name, district name, and population (number of electorates). A column of magnitude is optional.

    Value

    a prcalc object.

    See Also

    read_prcalc

    Examples

    library(PRcalc)

data(au_district_2010)

obj <- as_prcalc(au_district_2010,
                 l1   = "region",
                 l2   = "district",
                 p    = "electorates",
                 q    = "magnitude",
                 type = "district")
obj
